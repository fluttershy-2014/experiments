# 🐾 NekoLove: Episode 1 — «Ожидание»

Интерактивная 3D-сцена на **Three.js**, повествующая о коротком, но решающем моменте из жизни девушки-неко. Ты находишься в её комнате. На столе лежит записка, в зеркале — твоё отражение, а в кармане вибрирует телефон. До звонка, который изменит всё, осталось 60 секунд.

## 🛠 Что нового в V4.0

В этой версии мы сосредоточились на «полировке» игрового опыта и исправлении багов:

* **Camera Fix:** Модель персонажа больше не «залезает» в камеру при ходьбе и корректно отображается в зеркале.
* **Physics Update:** Увеличен хитбокс игрока до **0.9**, чтобы предотвратить прохождение сквозь стены и «провалы» внутрь зеркала.
* **Interactions:** Добавлена возможность прочитать записку на столе (клавиша **E**).
* **Audio System:** Реализован фоновый эмбиенс комнаты и входящий звонок в конце таймера.

## 🎮 Управление

| Действие | Клавиша / Управление |
| :--- | :--- |
| **Передвижение** | `W` `A` `S` `D` |
| **Обзор (360°)** | `Мышь` |
| **Взаимодействие** | `E` (прочитать записку) |
| **Захват курсора** | `ЛКМ` |
| **Закрыть записку** | `ЛКМ` |
| **Выход / Пауза** | `ESC` |
